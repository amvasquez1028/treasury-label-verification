# Branding

This demo applies **U.S. Web Design System (USWDS)–inspired** tokens without importing the full USWDS package:

- **Primary**: `#1a4480` (USWDS `primary-darker`)
- **Typography**: Public Sans (local `next/font` with system fallback)
- **Layout**: Treasury-style header/footer with department attribution

## Usage rules

- Keep header copy factual: "U.S. Department of the Treasury" is placeholder demo labeling
- Do not imply official production endorsement in user-facing error strings
- Maintain 4.5:1 contrast for body text on light backgrounds

See [CONTENT_STYLE.md](CONTENT_STYLE.md) for voice and tone.
